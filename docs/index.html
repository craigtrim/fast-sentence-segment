Fast Sentence Segment

Fast and efficient sentence segmentation using spaCy.
Handles abbreviations, quoted text, and multi-paragraph documents.

Python spaCy 3.5 MIT License

Quick Start

$ pip install fast-sentence-segment
from fast_sentence_segment import segment_text text = "Here is a Dr. who says something. And what else? I don't know." results = segment_text(text) # Returns: [['Here is a Dr. who says something.', 'And what else?', "I don't know."]]

Features

Paragraph-Aware

Returns sentences grouped by paragraph, preserving document structure for downstream processing.

Abbreviation Handling

Correctly handles "Dr.", "Mr.", "etc." and other abbreviations without false sentence splits.

Cached Processing

Built-in LRU cache for repeated text processing, improving performance on duplicate content.

Flexible Output

Get nested lists by paragraph or use flatten=True for a simple list of sentences.

Examples

Multi-Paragraph Text

text = """First paragraph here. It has two sentences. Second paragraph starts here. With multiple sentences.""" results = segment_text(text) # [ # ['First paragraph here.', 'It has two sentences.'], # ['Second paragraph starts here.', 'With multiple sentences.'] # ]

Flattened Output

results = segment_text(text, flatten=True) # ['First paragraph here.', 'It has two sentences.', 'Second paragraph starts here.', ...]
